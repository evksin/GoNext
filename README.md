# GoNext

Минимальное приложение на Expo Router + TypeScript с использованием React Native Paper.

## Возможности

- Экран Home с AppBar "GoNext"
- Текст "Привет, React Native Paper!"
- Кнопка "Нажми меня" со Snackbar "Кнопка нажата"

## Быстрый старт

1. Установить зависимости:
   `npm install --legacy-peer-deps`
2. Запустить Metro:
   `npx expo start`

## Скрипты

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
